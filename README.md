# Stock Portfolio Tracker
A simple Python project that helps users calculate the total value of their stock portfolio. Users can enter stock symbols and quantities, and the program computes the investment value based on predefined stock prices. The project also generates a portfolio report file for future reference.
## Features
- User-friendly console interface
- Predefined stock prices
- Portfolio value calculation
- Report generation using file handling
- Beginner-friendly Python project
## Technologies Used
- Python 3
